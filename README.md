# Skynet simple
## A project with game server use skynet.

## Project setup
```
make
```

### Start login server.
```
sh slogin.sh -i 1
```

### Start world server.
```
sh sworld.sh -i 1
```

### Start robot like mini client.
```
sh srobot.sh -i 1
```
